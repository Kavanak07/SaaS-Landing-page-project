# SaaS-Landing-page-project# 🚀 CampusFlow SaaS Landing Page

## 📌 Project Overview

CampusFlow is a modern SaaS landing page designed to capture user leads through an attractive and user-friendly interface. The project demonstrates frontend design skills along with basic backend integration for storing user data and sending verification emails.

This project focuses on creating a conversion-focused landing page that encourages users to sign up and interact with the platform.

---

## 🎯 Features

✨ Eye-catching landing page design  
✨ Responsive layout with modern UI  
✨ Signup form with validation  
✨ Stores user data  
✨ Sends welcome/verification email  
✨ Displays thank-you dashboard after signup  

---
